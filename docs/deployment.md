# Deployment Guide

The recommended deployment method is using helm.

Both the application container image and helm chart are published to the GitHub package registry:
- [ghcr.io/cturner8/kube-mcp](https://github.com/cturner8/kube-mcp/pkgs/container/kube-mcp): MCP container image
- [ghcr.io/cturner8/kube-mcp-chart](https://github.com/cturner8/kube-mcp/pkgs/container/kube-mcp-chart): Kubernetes deployment chart
